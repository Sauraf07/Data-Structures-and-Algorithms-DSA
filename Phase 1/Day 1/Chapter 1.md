# Before We Learn Python...

Let's answer one important question.

## 🤔 What is Programming?

Imagine you have a robot.

The robot is very powerful.

But...

It doesn't know what to do.

If you want the robot to make tea, you must tell it every single step.

Example:

```
Step 1:
Take a cup

Step 2:
Pour water

Step 3:
Boil water

Step 4:
Add tea

Step 5:
Add sugar

Step 6:
Serve
```

The robot follows exactly what you tell it.

It cannot think.

It cannot guess.

Computers are exactly like that.

A computer only follows instructions.

Those instructions are called

> **Program**

Writing those instructions is called

> **Programming**

---

# 🌍 Real Life Example

Imagine you order pizza.

You tell the delivery person

```
House Number

Street

City

Phone Number
```

If even one instruction is wrong...

They cannot deliver your pizza.

Similarly,

Computers also need exact instructions.

Programming is simply giving instructions to a computer.

We use Python to communicate with computers.

# 📜 History of Python

Python was created by

## Guido van Rossum

in

### 1991

His goal was very simple.

> "Programming should be easy to read and easy to write."

At that time,

many programming languages were difficult.

Python solved this problem.

# 🐍 Why is it Called Python?

Many beginners think

Python is named after the snake.

❌ Wrong.

It is actually named after

**Monty Python's Flying Circus**

which was a famous comedy TV show.

Guido liked that show.

So he named the language

Python.

Fun fact 😊

Python has nothing to do with snakes.

Let's understand.

---

# 🥇 Reason 1

Python is Easy

Example

C++

```cpp
#include<iostream>

using namespace std;

int main()
{
    cout<<"Hello";
}
```

Python

```python
print("Hello")
```

Which one looks easier?

Obviously Python.

---

# 🥇 Reason 2

Less Code

Suppose you want to print numbers from 1 to 10.

Python

```python
for i in range(1,11):
    print(i)
```

Simple.

Clean.

Readable.

---

# 🥇 Reason 3

Readable

Python looks almost like English.

Example

```python
if age >= 18:
    print("Eligible")
else:
    print("Not Eligible")
```

Even someone new to programming can understand it.

---

# 🥇 Reason 4

Huge Community

Millions of developers use Python.

That means

Whenever you get stuck,

someone has already solved that problem.

# 💼 Why Python for DSA?

Many students ask

Should I learn DSA in

C++

Java

or Python?

The answer depends on your goal.

If your goal is

- Placements
- Coding Interviews
- Competitive Programming (beginner to intermediate)
- AI
- Machine Learning
- Data Science

Python is an excellent choice.

Why?

Because

You spend less time writing syntax

and more time solving problems.

Example

Suppose you need a Queue.

In C++

You may need many lines of code.

In Python

```python
from collections import deque

queue = deque()
```

Done.
