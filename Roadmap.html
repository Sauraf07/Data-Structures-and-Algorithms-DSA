Q1
Two Sum — Find two numbers that add to target
Easy ⭐⭐⭐⭐⭐ Very High Amazon Google TCS NQT Infosys Wipro Zoho
▼
Problem Statement
Given an array of integers nums and an integer target, return the indices of the two numbers such that they add up to target. Assume exactly one solution exists. You may not use the same element twice.
🧠 Real-life example: Sochlo tum ek grocery store mein ho. List mein prices hain, aur tumhare paas exactly ₹100 hain. Kaunse 2 items milakaar exactly ₹100 banate hain?
Sample Input / Output
Input:nums = [2, 7, 11, 15], target = 9 Output:[0, 1] → nums[0] + nums[1] = 2 + 7 = 9 Input:nums = [3, 2, 4], target = 6 Output:[1, 2]
Brute Force Approach — O(n²)
Har element ke liye baaki sab elements check karo. Double loop lagao. Slow but simple.
Python — Brute Force
def two_sum_brute(nums, target):
    n = len(nums)
    for i in range(n):
        for j in range(i+1, n):
            if nums[i] + nums[j] == target:
                return [i, j]
    return []
✅ Optimal Approach — HashMap O(n)
HashMap mein complement store karo. Ek hi pass mein answer milega. Sochlo: agar target=9 aur current=2, toh mujhe 7 chahiye — kya 7 already mila hai? HashMap mein dekho!
Python — Optimal HashMap
def two_sum(nums, target):
    seen = {}  # value → index
    for i, num in enumerate(nums):
        complement = target - num
        if complement in seen:
            return [seen[complement], i]
        seen[num] = i
    return []

# Test
print(two_sum([2, 7, 11, 15], 9))  # [0, 1]
print(two_sum([3, 2, 4], 6))       # [1, 2]
Complexity
ApproachTimeSpace
Brute ForceO(n²)O(1)
HashMap (Optimal)O(n)O(n)
Edge Cases & Common Mistakes
⚠️ Common Mistake
Same element ko do baar use karna — e.g., [3,3] target=6, yahan index 0 aur 1 alag hain, yeh valid hai.
💡 Edge Case
Empty array → return []. Single element → return []. Negative numbers → HashMap handles them fine!
✅ Interview Tip
Pehle brute force explain karo, fir optimize karo. Interviewer process dekhna chahta hai.
⚠️ Don't Forget
Problem mein assume kiya hai exactly one solution hai. Agar multiple solutions ho toh all return karna padega.

Maximum Subarray Sum (Kadane's Algorithm)
Easy ⭐⭐⭐⭐⭐ Very High Amazon Microsoft TCS Infosys Wipro Nagarro
▼
Problem Statement
Given an array of integers, find the contiguous subarray (containing at least one number) which has the largest sum and return its sum.
🧠 Real-life: Stock market mein consecutive days ka maximum profit track karo. Kis period mein tumne sabse zyada kamaya?
Sample Input / Output
Input:[-2, 1, -3, 4, -1, 2, 1, -5, 4] Output:6 → Subarray [4, -1, 2, 1] = 6 Input:[-1, -2, -3] Output:-1 (all negative → pick max single element)
✅ Kadane's Algorithm — O(n)
Har position par decide karo: kya main current subarray continue karun ya nayi shuru karun? Agar current_sum + num < num → nayi start karo!
Python — Kadane's Algorithm
def max_subarray(nums):
    max_sum = nums[0]
    current_sum = nums[0]
    
    for num in nums[1:]:
        # Either extend current subarray or start fresh
        current_sum = max(num, current_sum + num)
        max_sum = max(max_sum, current_sum)
    
    return max_sum

# Bonus: Also return the subarray indices
def max_subarray_with_indices(nums):
    max_sum = current_sum = nums[0]
    start = end = temp_start = 0
    
    for i in range(1, len(nums)):
        if nums[i] > current_sum + nums[i]:
            current_sum = nums[i]
            temp_start = i
        else:
            current_sum += nums[i]
        if current_sum > max_sum:
            max_sum = current_sum
            start, end = temp_start, i
    
    return max_sum, nums[start:end+1]
Complexity
ApproachTimeSpace
Brute Force (all subarrays)O(n³)O(1)
Kadane's (Optimal)O(n)O(1)
Edge Cases & Interview Tips
⚠️ All Negative Array
Start with nums[0] as max, not 0. Agar 0 se start karo toh empty subarray count hoga!
✅ Follow-up Questions
Interviewer puch sakta hai: "What if we need the actual subarray?" → Track start/end indices.

3Sum — Find all unique triplets that sum to zero
Medium ⭐⭐⭐⭐ High Google Microsoft Amazon Adobe Flipkart
▼
Problem Statement
Given an integer array, return all the triplets [nums[i], nums[j], nums[k]] such that i≠j, i≠k, j≠k, and nums[i]+nums[j]+nums[k]==0. No duplicate triplets in answer.
Sample Input / Output
Input:[-1, 0, 1, 2, -1, -4] Output:[[-1,-1,2], [-1,0,1]]
✅ Optimal — Sort + Two Pointer O(n²)
Python — Sort + Two Pointer
def three_sum(nums):
    nums.sort()
    result = []
    
    for i in range(len(nums) - 2):
        # Skip duplicates for i
        if i > 0 and nums[i] == nums[i-1]:
            continue
        
        left, right = i+1, len(nums)-1
        
        while left < right:
            total = nums[i] + nums[left] + nums[right]
            if total == 0:
                result.append([nums[i], nums[left], nums[right]])
                # Skip duplicates
                while left < right and nums[left] == nums[left+1]:
                    left += 1
                while left < right and nums[right] == nums[right-1]:
                    right -= 1
                left += 1
                right -= 1
            elif total < 0:
                left += 1
            else:
                right -= 1
    
    return result
Complexity
ApproachTimeSpace
Brute Force (3 loops)O(n³)O(1)
Sort + Two PointerO(n²)O(1)
Q4
Rotate Array by K positions
Medium ⭐⭐⭐⭐ High TCSWiproCapgeminiInfoBeansNagarro
▼
Problem Statement
Given an array nums, rotate the array to the right by k steps, where k is non-negative. In-place with O(1) extra space.
Sample Input / Output
Input:nums = [1,2,3,4,5,6,7], k = 3 Output:[5,6,7,1,2,3,4]
✅ Optimal — Reverse Trick O(n)
3-step trick: (1) Reverse whole array, (2) Reverse first k elements, (3) Reverse remaining elements.
Python — Reverse Trick
def rotate(nums, k):
    n = len(nums)
    k = k % n  # Handle k > n
    
    def reverse(l, r):
        while l < r:
            nums[l], nums[r] = nums[r], nums[l]
            l += 1; r -= 1
    
    reverse(0, n-1)    # Step 1: Reverse all
    reverse(0, k-1)    # Step 2: Reverse first k
    reverse(k, n-1)    # Step 3: Reverse rest
    return nums

# Easy but O(n) space approach:
def rotate_easy(nums, k):
    k = k % len(nums)
    return nums[-k:] + nums[:-k]
Complexity
ApproachTimeSpace
Extra ArrayO(n)O(n)
Reverse TrickO(n)O(1)

Trapping Rain Water
Hard ⭐⭐⭐⭐⭐ Very High AmazonGoogleMicrosoftAdobeUber
▼
Problem Statement
Given n non-negative integers representing an elevation map where the width of each bar is 1, compute how much water it can trap after raining.
🧠 Real-life: Pahad jaise elevations ke beech barish ka paani kahan ruk sakta hai? Har bar ke upar kitna paani aayega?
Sample Input / Output
Input:height = [0,1,0,2,1,0,1,3,2,1,2,1] Output:6 Input:height = [4,2,0,3,2,5] Output:9
✅ Optimal — Two Pointer O(n) Time, O(1) Space
Python — Two Pointer (Best)
def trap(height):
    if not height: return 0
    
    left, right = 0, len(height) - 1
    left_max = right_max = 0
    water = 0
    
    while left < right:
        if height[left] < height[right]:
            if height[left] >= left_max:
                left_max = height[left]
            else:
                water += left_max - height[left]
            left += 1
        else:
            if height[right] >= right_max:
                right_max = height[right]
            else:
                water += right_max - height[right]
            right -= 1
    
    return water

# Key Insight: Water at index i = min(max_left, max_right) - height[i]

ApproachTimeSpace

Precompute ArraysO(n)O(n)

Valid Palindrome
Easy ⭐⭐⭐⭐⭐ Very High TCSInfosysWiproFacebookInfoBeans
▼
Problem Statement
A phrase is a palindrome if, after converting all uppercase letters into lowercase letters and removing all non-alphanumeric characters, it reads the same forward and backward.
Sample Input / Output
Input:"A man, a plan, a canal: Panama" Output:True Input:"race a car" Output:False
✅ Optimal — Two Pointer
Python
def is_palindrome(s):
    # Clean: keep only alphanumeric, lowercase
    cleaned = ''.join(c.lower() for c in s if c.isalnum())
    return cleaned == cleaned[::-1]

# Two-pointer (O(1) space conceptually):
def is_palindrome_tp(s):
    left, right = 0, len(s) - 1
    while left < right:
        while left < right and not s[left].isalnum():
            left += 1
        while left < right and not s[right].isalnum():
            right -= 1
        if s[left].lower() != s[right].lower():
            return False
        left += 1; right -= 1
    return True

ApproachTimeSpace

Valid Anagram
Easy ⭐⭐⭐⭐⭐ Very High TCSCognizantAmazonZoho
▼
Problem Statement
Given two strings s and t, return true if t is an anagram of s, and false otherwise. An anagram uses same characters same number of times.
🧠 "listen" aur "silent" — dono same letters se bane hain! Anagram hai!
Sample Input / Output
Input:s = "anagram", t = "nagaram" Output:True Input:s = "rat", t = "car" Output:False
✅ Solutions
Python — Multiple Approaches
from collections import Counter

# Method 1: Counter (Pythonic)
def is_anagram_1(s, t):
    return Counter(s) == Counter(t)

# Method 2: Sort
def is_anagram_2(s, t):
    return sorted(s) == sorted(t)  # O(n log n)

# Method 3: Frequency Array (Best for interviews)
def is_anagram_3(s, t):
    if len(s) != len(t): return False
    count = [0] * 26
    for c in s: count[ord(c) - ord('a')] += 1
    for c in t: count[ord(c) - ord('a')] -= 1
    return all(x == 0 for x in count)

Longest Substring Without Repeating Characters
Medium ⭐⭐⭐⭐⭐ Very High AmazonMicrosoftGoogleAdobeRazorpayPhonePe
▼
Problem Statement
Given a string s, find the length of the longest substring without repeating characters.
🧠 "pwwkew" → longest is "wke" = 3. Sliding window technique ka classic example!
Sample Input / Output
Input:"abcabcbb" → Output: 3 ("abc") Input:"bbbbb" → Output: 1 ("b") Input:"pwwkew" → Output: 3 ("wke")
✅ Sliding Window + HashMap O(n)
Python — Sliding Window
def length_of_longest_substring(s):
    char_index = {}  # char → last seen index
    max_len = 0
    left = 0  # window start
    
    for right, char in enumerate(s):
        if char in char_index and char_index[char] >= left:
            # Shrink window from left
            left = char_index[char] + 1
        char_index[char] = right
        max_len = max(max_len, right - left + 1)
    
    return max_len

ApproachTimeSpace

Reverse a Linked List
Easy ⭐⭐⭐⭐⭐ Very High TCSInfosysWiproAmazonMicrosoft
▼
Problem Statement
Given the head of a singly linked list, reverse the list, and return the reversed list.
🧠 Train wagons ki tarah sochlo — 1→2→3→4→5 ko 5→4→3→2→1 karna hai. Ek-ek wagon ka direction badlo!
Sample Input / Output
Input:1 → 2 → 3 → 4 → 5 Output:5 → 4 → 3 → 2 → 1
✅ Iterative + Recursive Solutions
Python — Both Approaches
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

# Iterative (Preferred in interviews)
def reverse_list(head):
    prev = None
    current = head
    
    while current:
        next_node = current.next  # Save next
        current.next = prev       # Reverse pointer
        prev = current            # Move prev forward
        current = next_node       # Move current forward
    
    return prev  # New head

# Recursive
def reverse_list_rec(head):
    if not head or not head.next:
        return head
    new_head = reverse_list_rec(head.next)
    head.next.next = head
    head.next = None
    return new_head
Complexity
ApproachTimeSpace

Detect Cycle in Linked List (Floyd's Tortoise & Hare)
Medium ⭐⭐⭐⭐⭐ Very High AmazonGoogleMicrosoftPaytmSwiggy
▼
Problem Statement
Given head of a linked list, determine if the linked list has a cycle. Can you do it with O(1) memory?
🧠 Circular race track! Ek runner fast (2 step), ek slow (1 step). Agar track circular hai toh fast wala slow ko zaroor pakad lega!
Sample Input / Output
Input:3 → 2 → 0 → -4 → (back to 2) Output:True (cycle exists at node 2) Input:1 → 2 → None Output:False
✅ Floyd's Algorithm
Python — Floyd's Tortoise & Hare
def has_cycle(head):
    slow = fast = head
    
    while fast and fast.next:
        slow = slow.next        # 1 step
        fast = fast.next.next   # 2 steps
        if slow == fast:
            return True
    
    return False

# Bonus: Find the start of cycle
def detect_cycle_start(head):
    slow = fast = head
    while fast and fast.next:
        slow = slow.next
        fast = fast.next.next
        if slow == fast:
            # Reset one pointer to head
            slow = head
            while slow != fast:
                slow = slow.next
                fast = fast.next
            return slow  # Cycle start node
    return None
Complexity
ApproachTimeSpace
HashSetO(n)O(n)
Floyd's (Optimal)O(n)O(1)

Given a string s containing just the characters '(', ')', '{', '}', '[' and ']', determine if the input string is valid. Each open bracket must be closed in correct order.
🧠 HTML tags ki tarah! <div><p></p></div> valid, lekin <div><p></div></p> invalid!
Sample Input / Output

def is_valid(s):
    stack = []
    mapping = {')': '(', '}': '{', ']': '['}
    
    for char in s:
        if char in mapping:
            # Closing bracket
            top = stack.pop() if stack else '#'
            if mapping[char] != top:
                return False
        else:
            # Opening bracket
            stack.append(char)
    
    return not stack  # Stack must be empty

Q12
Next Greater Element (Monotonic Stack)
Medium ⭐⭐⭐⭐ High AmazonPaytmSamsungQualcomm
▼
Problem Statement
For each element, find the next greater element to its right. If no such element exists, output -1.
Sample Input / Output
Input:[4, 5, 2, 10, 8] Output:[5, 10, 10, -1, -1]
✅ Monotonic Stack O(n)
Python — Monotonic Stack
def next_greater(nums):
    result = [-1] * len(nums)
    stack = []  # stores indices
    
    for i, num in enumerate(nums):
        # While stack not empty and current > top of stack
        while stack and nums[stack[-1]] < num:
            idx = stack.pop()
            result[idx] = num
        stack.append(i)
    
    return result

Binary Tree Inorder / Preorder / Postorder Traversal
Easy ⭐⭐⭐⭐⭐ Very High TCSInfosysAmazonMicrosoftYash Technologies
▼
Problem Statement
Implement Inorder (Left-Root-Right), Preorder (Root-Left-Right), and Postorder (Left-Right-Root) traversal of a Binary Tree — both recursive and iterative.
🧠 Family tree sochlo: Inorder = siblings pehle, fir aap, fir right side. Preorder = aap pehle, fir baaki. BST ka Inorder always gives sorted output!
✅ All Traversals — Recursive
Python — Tree Traversals
class TreeNode:
    def __init__(self, val=0):
        self.val = val
        self.left = self.right = None

# Inorder: Left → Root → Right (BST → sorted order)
def inorder(root):
    if not root: return []
    return inorder(root.left) + [root.val] + inorder(root.right)

# Preorder: Root → Left → Right
def preorder(root):
    if not root: return []
    return [root.val] + preorder(root.left) + preorder(root.right)

# Postorder: Left → Right → Root
def postorder(root):
    if not root: return []
    return postorder(root.left) + postorder(root.right) + [root.val]

# Level Order (BFS) — Very common!
from collections import deque
def level_order(root):
    if not root: return []
    result, queue = [], deque([root])
    while queue:
        level = []
        for _ in range(len(queue)):
            node = queue.popleft()
            level.append(node.val)
            if node.left: queue.append(node.left)
            if node.right: queue.append(node.right)
        result.append(level)
    return result

Easy ⭐⭐⭐⭐⭐ Very High AmazonTCSInfosysWiproCognizant
▼
✅ Recursive DFS
Python
def max_depth(root):
    if not root: return 0
    return 1 + max(max_depth(root.left), max_depth(root.right))

# Iterative BFS (Level by Level)
def max_depth_bfs(root):
    if not root: return 0
    queue, depth = deque([root]), 0
    while queue:
        depth += 1
        for _ in range(len(queue)):
            node = queue.popleft()
            if node.left: queue.append(node.left)
            if node.right: queue.append(node.right)
    return depth
Q15
Lowest Common Ancestor of BST
Medium ⭐⭐⭐⭐⭐ Very High AmazonGoogleMicrosoftFacebook
▼
Problem Statement
Given a BST, find the lowest common ancestor (LCA) of two given nodes p and q. LCA is the lowest node in the tree that has both p and q as descendants.
✅ BST Property Solution
Python — BST LCA
def lca_bst(root, p, q):
    # In BST: if both < root → go left, both > root → go right
    if p.val < root.val and q.val < root.val:
        return lca_bst(root.left, p, q)
    elif p.val > root.val and q.val > root.val:
        return lca_bst(root.right, p, q)
    else:
        return root  # Split point = LCA

# General Binary Tree LCA (not BST)
def lca_bt(root, p, q):
    if not root or root == p or root == q:
        return root
    left = lca_bt(root.left, p, q)
    right = lca_bt(root.right, p, q)
    return root if left and right else left or right

Q16
Fibonacci — Memoization vs Tabulation
Easy ⭐⭐⭐⭐⭐ Very High TCSInfosysWiproAmazonAll Companies
▼
Problem Statement
Calculate the nth Fibonacci number. F(0)=0, F(1)=1, F(n)=F(n-1)+F(n-2). This is the gateway to understanding DP — from exponential to linear!
Sample Input / Output
Input:n = 10 → Output: 55 Input:n = 6 → Output: 8 (0,1,1,2,3,5,8)
✅ All 4 Approaches
Python — 4 Approaches
# 1. Naive Recursion — O(2^n) ❌ Very Slow
def fib_naive(n):
    if n <= 1: return n
    return fib_naive(n-1) + fib_naive(n-2)

# 2. Memoization (Top-Down DP) — O(n)
def fib_memo(n, memo={}):
    if n in memo: return memo[n]
    if n <= 1: return n
    memo[n] = fib_memo(n-1, memo) + fib_memo(n-2, memo)
    return memo[n]

# 3. Tabulation (Bottom-Up DP) — O(n) time, O(n) space
def fib_tab(n):
    if n <= 1: return n
    dp = [0] * (n+1)
    dp[1] = 1
    for i in range(2, n+1):
        dp[i] = dp[i-1] + dp[i-2]
    return dp[n]

# 4. Space Optimized — O(n) time, O(1) space ✅ Best
def fib_opt(n):
    if n <= 1: return n
    a, b = 0, 1
    for _ in range(n-1):
        a, b = b, a+b
    return b
Q17
0/1 Knapsack Problem
Medium ⭐⭐⭐⭐⭐ Very High AmazonMicrosoftTCS NinjaPersistentFlipkart
▼
Problem Statement
Given n items with weights and values, and a knapsack with capacity W, find the maximum value you can put in the knapsack. Each item can be taken at most once (0 or 1).
🧠 Thief ki tarah sochlo! Bag mein limited weight uthaa sakte ho — kaunse items utha lo maximum value ke liye?
Sample Input / Output
Input:weights=[2,3,4,5], values=[3,4,5,6], W=5 Output:7 (items with weight 2 and 3: value 3+4=7)
✅ DP Table Solution O(n×W)
Python — 0/1 Knapsack DP
def knapsack(weights, values, W):
    n = len(weights)
    # dp[i][w] = max value using first i items with capacity w
    dp = [[0] * (W+1) for _ in range(n+1)]
    
    for i in range(1, n+1):
        for w in range(W+1):
            # Don't take item i
            dp[i][w] = dp[i-1][w]
            # Take item i (if it fits)
            if weights[i-1] <= w:
                dp[i][w] = max(dp[i][w], 
                    dp[i-1][w-weights[i-1]] + values[i-1])
    
    return dp[n][W]
Complexity
ApproachTimeSpace
Recursion (Brute)O(2^n)O(n)
MemoizationO(n×W)O(n×W)
Tabulation (DP)O(n×W)O(n×W)
Q18
Longest Common Subsequence (LCS)
Medium ⭐⭐⭐⭐⭐ Very High AmazonGoogleFlipkartOracleZoho
▼
Problem Statement
Given two strings, find the length of their Longest Common Subsequence. A subsequence keeps order but elements don't need to be contiguous.
Sample Input / Output
Input:s1 = "ABCBDAB", s2 = "BDCAB" Output:4 → LCS = "BCAB" or "BDAB"
✅ DP Solution O(m×n)
Python — LCS DP
def lcs(s1, s2):
    m, n = len(s1), len(s2)
    dp = [[0] * (n+1) for _ in range(m+1)]
    
    for i in range(1, m+1):
        for j in range(1, n+1):
            if s1[i-1] == s2[j-1]:
                dp[i][j] = dp[i-1][j-1] + 1
            else:
                dp[i][j] = max(dp[i-1][j], dp[i][j-1])
    
    return dp[m][n]

def coin_change(coins, amount):
    dp = [float('inf')] * (amount + 1)
    dp[0] = 0  # 0 coins for amount 0
    
    for coin in coins:
        for amt in range(coin, amount + 1):
            dp[amt] = min(dp[amt], dp[amt - coin] + 1)
    
    return dp[amount] if dp[amount] != float('inf') else -1

# O(n²) DP
def lis_dp(nums):
    n = len(nums)
    dp = [1] * n  # each element is LIS of length 1
    for i in range(1, n):
        for j in range(i):
            if nums[j] < nums[i]:
                dp[i] = max(dp[i], dp[j] + 1)
    return max(dp)

import bisect
def lis_optimal(nums):
    tails = []
    for num in nums:
        pos = bisect.bisect_left(tails, num)
        if pos == len(tails):
            tails.append(num)
        else:
            tails[pos] = num
    return len(tails)

🧠 BFS = level by level (pehle saare friends, fir friends-of-friends) — Facebook friend suggestions! DFS = ek path tak jaao (maze solving). Swiggy delivery route optimization!

Python — BFS & DFS
from collections import deque

# BFS — Queue based
def bfs(graph, start):
    visited = {start}
    queue = deque([start])
    order = []
    
    while queue:
        node = queue.popleft()
        order.append(node)
        for neighbor in graph[node]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
    return order

# DFS — Recursive
def dfs(graph, node, visited=None):
    if visited is None: visited = set()
    visited.add(node)
    order = [node]
    for neighbor in graph[node]:
        if neighbor not in visited:
            order.extend(dfs(graph, neighbor, visited))
    return order

# DFS — Iterative (Stack)
def dfs_iterative(graph, start):
    visited, stack, order = set(), [start], []
    while stack:
        node = stack.pop()
        if node not in visited:
            visited.add(node)
            order.append(node)
            stack.extend(graph[node])
    return order

Given a 2D grid of '1' (land) and '0' (water), count the number of islands. An island is surrounded by water and formed by connecting adjacent lands horizontally/vertically.
Sample Input / Output
Input: 11110 11010 11000 00000 Output:1 island 11000 11000 00100 00011 Output:3 islands
✅ DFS Flood Fill

def num_islands(grid):
    if not grid: return 0
    rows, cols = len(grid), len(grid[0])
    count = 0
    
    def dfs(r, c):
        if r < 0 or r >= rows or c < 0 or c >= cols:
            return
        if grid[r][c] != '1': return
        grid[r][c] = '#'  # Mark as visited
        dfs(r+1,c); dfs(r-1,c); dfs(r,c+1); dfs(r,c-1)
    
    for r in range(rows):
        for c in range(cols):
            if grid[r][c] == '1':
                dfs(r, c)
                count += 1
    return count

def merge_sort(arr):
    if len(arr) <= 1: return arr
    
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i]); i += 1
        else:
            result.append(right[j]); j += 1
    
    result.extend(left[i:])
    result.extend(right[j:])
    return result

# Complexity: Time O(n log n), Space O(n)
# Stable Sort — equal elements maintain relative order

# Standard Binary Search O(log n)
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = left + (right - left) // 2  # Avoids overflow
        if arr[mid] == target: return mid
        elif arr[mid] < target: left = mid + 1
        else: right = mid - 1
    return -1

# Search in Rotated Sorted Array (VERY COMMON)
def search_rotated(nums, target):
    left, right = 0, len(nums) - 1
    while left <= right:
        mid = (left + right) // 2
        if nums[mid] == target: return mid
        # Left half is sorted
        if nums[left] <= nums[mid]:
            if nums[left] <= target < nums[mid]:
                right = mid - 1
            else:
                left = mid + 1
        else:  # Right half is sorted
            if nums[mid] < target <= nums[right]:
                left = mid + 1
            else:
                right = mid - 1
    return -1

Maximum Sum of Subarray of Size K

Problem Statement
Given an array and integer k, find the maximum sum of any contiguous subarray of size k.
✅ Sliding Window O(n)
Python — Sliding Window
def max_sum_subarray(arr, k):
    # Initialize first window
    window_sum = sum(arr[:k])
    max_sum = window_sum
    
    # Slide the window
    for i in range(k, len(arr)):
        window_sum += arr[i] - arr[i-k]  # Add new, remove old
        max_sum = max(max_sum, window_sum)
    
    return max_sum
# Time: O(n), Space: O(1) — Much better than O(n²) brute!

Python — Subsets via Backtracking
def subsets(nums):
    result = []
    
    def backtrack(start, current):
        result.append(current[:])  # Add copy of current subset
        for i in range(start, len(nums)):
            current.append(nums[i])
            backtrack(i+1, current)
            current.pop()  # Undo choice (backtrack)
    
    backtrack(0, [])
    return result

# nums=[1,2,3] → [[], [1], [1,2], [1,2,3], [1,3], [2], [2,3], [3]]
# Total subsets = 2^n
Q27
N-Queens Problem
Hard ⭐⭐⭐⭐ High GoogleAmazonMicrosoftAdobe
▼
Problem Statement
Place n queens on an n×n chessboard such that no two queens attack each other. Return all valid placements.
✅ Backtracking
Python — N-Queens
def solve_n_queens(n):
    results = []
    cols = set()
    diag1 = set()  # r - c
    diag2 = set()  # r + c
    board = ['.' * n for _ in range(n)]
    
    def backtrack(row):
        if row == n:
            results.append(board[:])
            return
        for col in range(n):
            if col in cols or (row-col) in diag1 or (row+col) in diag2:
                continue
            # Place queen
            cols.add(col); diag1.add(row-col); diag2.add(row+col)
            board[row] = '.'*col + 'Q' + '.'*(n-col-1)
            backtrack(row + 1)
            # Remove queen
            cols.remove(col); diag1.remove(row-col); diag2.remove(row+col)
    
    backtrack(0)
    return results

import heapq

# Method 1: nlargest (Pythonic)
def k_largest_1(nums, k):
    return heapq.nlargest(k, nums)  # O(n log k)

# Method 2: Min-heap of size k
def k_largest_2(nums, k):
    heap = nums[:k]
    heapq.heapify(heap)  # Min-heap
    
    for num in nums[k:]:
        if num > heap[0]:
            heapq.heapreplace(heap, num)
    
    return sorted(heap, reverse=True)

# Kth Largest Element (Very Common Interview Q)
def kth_largest(nums, k):
    return heapq.nlargest(k, nums)[-1]  # O(n log k)

def activity_selection(start, finish):
    # Sort by finish time (Greedy choice)
    activities = sorted(zip(start, finish), key=lambda x: x[1])
    
    selected = [activities[0]]
    last_finish = activities[0][1]
    
    for s, f in activities[1:]:
        if s >= last_finish:  # No overlap
            selected.append((s, f))
            last_finish = f
    
    return len(selected), selected

# Meeting Rooms II — Minimum rooms needed
def min_meeting_rooms(intervals):
    if not intervals: return 0
    starts = sorted(i[0] for i in intervals)
    ends = sorted(i[1] for i in intervals)
    rooms = used = 0
    e = 0
    for s in starts:
        if s < ends[e]:
            rooms += 1
        else:
            e += 1
        used = max(used, rooms)
    return used

def single_number(nums):
    result = 0
    for num in nums:
        result ^= num  # XOR: a^a=0, a^0=a
    return result

Implement Trie (Prefix Tree)
Medium ⭐⭐⭐⭐ High AmazonGoogleMicrosoftAtlassian

Problem Statement
Implement a Trie with insert, search, and startsWith methods. Used in autocomplete, spell checkers, IP routing.
🧠 Google search bar ka autocomplete! Jab tum "Ind" type karte ho, "India", "Indonesia", "Indore" suggest hoti hai — Trie se!
✅ Trie Implementation
Python — Trie
class TrieNode:
    def __init__(self):
        self.children = {}
        self.is_end = False

class Trie:
    def __init__(self):
        self.root = TrieNode()
    
    def insert(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                node.children[char] = TrieNode()
            node = node.children[char]
        node.is_end = True
    
    def search(self, word):
        node = self.root
        for char in word:
            if char not in node.children:
                return False
            node = node.children[char]
        return node.is_end
    
    def starts_with(self, prefix):
        node = self.root
        for char in prefix:
            if char not in node.children:
                return False
            node = node.children[char]
        return True

# Time: O(L) per operation where L = word length

from abc import ABC, abstractmethod
from datetime import datetime

# Abstract Base Class (Abstraction)
class Account(ABC):
    def __init__(self, account_no, owner, balance=0):
        self.__account_no = account_no    # Private (Encapsulation)
        self.__owner = owner
        self.__balance = balance
        self.__transactions = []
    
    @property
    def balance(self):
        return self.__balance
    
    def deposit(self, amount):
        if amount <= 0:
            raise ValueError("Amount must be positive")
        self.__balance += amount
        self.__transactions.append({'type':'deposit','amt':amount})
        return self.__balance
    
    @abstractmethod
    def withdraw(self, amount):
        pass  # Different behavior per account type
    
    def __str__(self):
        return f"Account({self.__owner}, ₹{self.__balance})"

# Inheritance
class SavingsAccount(Account):
    MIN_BALANCE = 500
    
    def withdraw(self, amount):
        if self.balance - amount < self.MIN_BALANCE:
            raise ValueError(f"Minimum balance ₹{self.MIN_BALANCE} required")
        self._Account__balance -= amount
        return self.balance

class CurrentAccount(Account):
    OVERDRAFT_LIMIT = 10000
    
    def withdraw(self, amount):  # Polymorphism
        if self.balance - amount < -self.OVERDRAFT_LIMIT:
            raise ValueError("Overdraft limit exceeded")
        self._Account__balance -= amount
        return self.balance

Problem Statement
Design a data structure that follows Least Recently Used (LRU) cache eviction policy. Implement get(key) and put(key, value) — both O(1) time.
🧠 Browser ka back button! Jo page sabse pehle visit kiya tha aur abhi koi kaam nahi, woh pehle hatega. CPU cache bhi same principle use karta hai!
✅ HashMap + Doubly Linked List
Python — LRU Cache
from collections import OrderedDict

# Simple approach using OrderedDict
class LRUCache:
    def __init__(self, capacity):
        self.capacity = capacity
        self.cache = OrderedDict()
    
    def get(self, key):
        if key not in self.cache:
            return -1
        self.cache.move_to_end(key)  # Mark as recently used
        return self.cache[key]
    
    def put(self, key, value):
        if key in self.cache:
            self.cache.move_to_end(key)
        self.cache[key] = value
        if len(self.cache) > self.capacity:
            self.cache.popitem(last=False)  # Remove LRU (first)

# Interviewer may ask for custom DLL implementation:
class DLLNode:
    def __init__(self, key, val):
        self.key, self.val = key, val
        self.prev = self.next = None

class LRUCacheManual:
    def __init__(self, capacity):
        self.cap = capacity
        self.cache = {}
        # Dummy head (LRU end) and tail (MRU end)
        self.head = DLLNode(0, 0)
        self.tail = DLLNode(0, 0)
        self.head.next = self.tail
        self.tail.prev = self.head
    
    def _remove(self, node):
        node.prev.next = node.next
        node.next.prev = node.prev
    
    def _insert_tail(self, node):  # MRU position
        node.prev = self.tail.prev
        node.next = self.tail
        self.tail.prev.next = node
        self.tail.prev = node
    
    def get(self, key):
        if key in self.cache:
            self._remove(self.cache[key])
            self._insert_tail(self.cache[key])
            return self.cache[key].val
        return -1
    
    def put(self, key, value):
        if key in self.cache:
            self._remove(self.cache[key])
        node = DLLNode(key, value)
        self.cache[key] = node
        self._insert_tail(node)
        if len(self.cache) > self.cap:
            lru = self.head.next
            self._remove(lru)
            del self.cache[lru.key]

Producer-Consumer Problem (Threading)

✅ Python Threading Solution
Python — Producer Consumer
import threading
import queue
import time

class ProducerConsumer:
    def __init__(self, buffer_size=5):
        self.buffer = queue.Queue(maxsize=buffer_size)
        self.lock = threading.Lock()
    
    def producer(self, items):
        for item in items:
            self.buffer.put(item)  # Blocks if full
            with self.lock:
                print(f"Produced: {item}")
            time.sleep(0.1)
    
    def consumer(self):
        while True:
            item = self.buffer.get()  # Blocks if empty
            if item is None: break  # Poison pill
            with self.lock:
                print(f"Consumed: {item}")
            self.buffer.task_done()

# Key Concepts:
# Race Condition: Multiple threads modify shared data simultaneously
# Mutex/Lock: Only one thread at a time
# Deadlock: Thread A waits for B, B waits for A → forever blocked
# Semaphore: Limits number of threads accessing a resource
# GIL in Python: Global Interpreter Lock — true parallelism needs multiprocessing

CS Fundamentals Asked in Interviews
QuestionQuick Answer
Process vs Thread?Process = independent program with its own memory. Thread = lightweight unit within a process, shares memory.
Stack vs Heap Memory?Stack = function calls, local variables (LIFO, auto-managed). Heap = dynamic allocation (malloc/new), manual or GC.
What is Deadlock?Circular wait — Thread A holds R1 waits for R2, Thread B holds R2 waits for R1. Prevention: ordering, timeouts.
ACID Properties?Atomicity, Consistency, Isolation, Durability — database transaction guarantees.
REST vs SOAP?REST: stateless, JSON, HTTP verbs. SOAP: XML, more rigid, has WSDL. REST preferred for modern APIs.
TCP vs UDP?TCP: reliable, ordered, connection-based (HTTP, FTP). UDP: fast, no guarantee (video streaming, gaming, DNS).
Normalization?1NF (atomic values) → 2NF (no partial dependency) → 3NF (no transitive dependency). Reduces redundancy.
What is an Index in DB?Data structure (B-tree usually) that speeds up SELECT but slows INSERT/UPDATE. Like book's table of contents.

Q56
Word Break Problem
Medium ⭐⭐⭐⭐ High GoogleAmazonFacebook
▼
Problem Statement
Given a string s and a dictionary of strings wordDict, return true if s can be segmented into space-separated sequence of dictionary words.
Sample Input / Output
Input:s = "leetcode", wordDict = ["leet","code"] → True Input:s = "applepenapple", wordDict = ["apple","pen"] → True
✅ DP Solution
Python — Word Break DP
def word_break(s, wordDict):
    word_set = set(wordDict)
    n = len(s)
    dp = [False] * (n + 1)
    dp[0] = True  # Empty string can always be segmented
    
    for i in range(1, n+1):
        for j in range(i):
            if dp[j] and s[j:i] in word_set:
                dp[i] = True
                break
    
    return dp[n]
Q57
Find Median from Data Stream
Hard ⭐⭐⭐⭐ High AmazonGoogleMicrosoft
▼
✅ Two Heaps Approach
Python — Two Heaps
import heapq

class MedianFinder:
    def __init__(self):
        self.small = []  # Max-heap (lower half) — negate for Python
        self.large = []  # Min-heap (upper half)
    
    def add_num(self, num):
        heapq.heappush(self.small, -num)
        # Balance: all in small <= all in large
        if self.small and self.large and (-self.small[0] > self.large[0]):
            heapq.heappush(self.large, -heapq.heappop(self.small))
        # Size balance
        if len(self.small) > len(self.large) + 1:
            heapq.heappush(self.large, -heapq.heappop(self.small))
        elif len(self.large) > len(self.small):
            heapq.heappush(self.small, -heapq.heappop(self.large))
    
    def find_median(self):
        if len(self.small) > len(self.large):
            return -self.small[0]
        return (-self.small[0] + self.large[0]) / 2

Sudoku Solver

✅ Backtracking Solution
Python — Sudoku Solver
def solve_sudoku(board):
    def is_valid(row, col, num):
        n = str(num)
        # Check row, col, 3x3 box
        box_r, box_c = 3*(row//3), 3*(col//3)
        return (n not in board[row] and
                n not in [board[i][col] for i in range(9)] and
                n not in [board[box_r+i][box_c+j] for i in range(3) for j in range(3)])
    
    def solve():
        for i in range(9):
            for j in range(9):
                if board[i][j] == '.':
                    for num in range(1, 10):
                        if is_valid(i, j, num):
                            board[i][j] = str(num)
                            if solve(): return True
                            board[i][j] = '.'  # Backtrack
                    return False
        return True
    
    solve()
Q59
Segment Tree — Range Sum Queries
Hard ⭐⭐⭐ Medium AmazonGoogleCodeforces-style
▼
Problem Statement
Build a data structure for range sum queries and point updates in O(log n) time.
✅ Segment Tree Implementation
Python — Segment Tree
class SegmentTree:
    def __init__(self, nums):
        self.n = len(nums)
        self.tree = [0] * (4 * self.n)
        self.build(nums, 0, 0, self.n-1)
    
    def build(self, nums, node, start, end):
        if start == end:
            self.tree[node] = nums[start]
        else:
            mid = (start + end) // 2
            self.build(nums, 2*node+1, start, mid)
            self.build(nums, 2*node+2, mid+1, end)
            self.tree[node] = self.tree[2*node+1] + self.tree[2*node+2]
    
    def query(self, l, r, node=0, start=0, end=None):
        if end is None: end = self.n-1
        if r < start or l > end: return 0
        if l <= start and end <= r: return self.tree[node]
        mid = (start + end) // 2
        return (self.query(l, r, 2*node+1, start, mid) +
                self.query(l, r, 2*node+2, mid+1, end))

# Query: O(log n), Build: O(n), Space: O(n)
Q60
Merge K Sorted Linked Lists
Hard ⭐⭐⭐⭐⭐ Very High AmazonGoogleMicrosoftUber
▼
✅ Min-Heap Approach O(N log K)
Python — Merge K Lists with Heap
import heapq

def merge_k_lists(lists):
    heap = []
    dummy = ListNode(0)
    current = dummy
    
    # Initialize heap with first node of each list
    for i, node in enumerate(lists):
        if node:
            heapq.heappush(heap, (node.val, i, node))
    
    while heap:
        val, i, node = heapq.heappop(heap)
        current.next = node
        current = current.next
        if node.next:
            heapq.heappush(heap, (node.next.val, i, node.next))
    
    return dummy.next

# N = total nodes, K = number of lists
# Time: O(N log K), Space: O(K)
